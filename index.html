Tipos de etiquetas

TIPOS DE ETIQUETAS

En HTML y CSS tenemos diferente tipos de etiquetas cada uno con un propósito en particular una etiqueta se puede componerse de dos maneras.

ESTRUCTURA DE ETIQUETAS
Etiquetas que solo se apertura Etiquetas que tienen una apertura y un cierre
Este tipo de etiqueta comúnmente se utilizan para indicar una dirección u origen se componen por argumento u opciones tales como src href. Este tipo de etiquetas comúnmente se utilizan para encerrar contenidos tales como texto, img, entre otros.

Lista de etiquetas dirigidas al texto:

  • <p></p> Se utiliza para colocar texto.
  • <h1></h1> a <h6></h6> Se utiliza para colocar un titulo directo en html.
  • <u></u> Se utiliza para que el texto salga subrayado.
  • <ol></ol> Se utiliza para crear listas enumeradas
  • <ul></ul> Se utiliza para crear listas con viñetas
  • <li></li> Se utiliza para definir un articulo de una lista.
  • <br> Se utiliza para representar un salto de linea

Listado de etiquetas dirigidas a objetos:

  • <img> Se utiliza para colocar una imagen.
  • <header> Representa un grupo de ayudas introductorias o de navegación.
  • <table></table> Se utiliza para colocar una tabla
  • <tr></tr> Se utiliza para colocar una fila.
  • <td></td>Se utiliza para colocar una celda.
  • <form></form> Se utiliza para colocar un formulario.
  • <a></a> Se utiliza para colocar un enlace.

Listado de propiedades css dirigidas a texto:

  • font-family; Se utiliza para colocar la fuente de la letra.
  • color; Se utiliza para seleccionar el color del texto.
  • font-size; Se utiliza para escoger el tamaño de letra.
  • text-aling; Se utiliza para seleccionar en que posicion queremos el texto.
  • margin; Se utiliza para colocar cuatro márgenes.
  • border-Style; Se utiliza para seleccionar el estilo del borde.
  • text-decoration; Especifica la decoracion del texto.
  • border-color;Especifica el color del borde del elemento.
  • background-color; Especifica el color del fondo del elemento.

Listado de propiedades dirigidas a objetos:

  • margin; Se utiliza para colocar cuatro márgenes.
  • width; Especifica la anchura del area de contenido de un elemento.
  • height;Especifica la altura del area de contenido de un elemento.
  • background-color; Especifica el color del fondo del elemento.
  • background-image; Se utiliza para el fondo de la imagen.
